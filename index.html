結合エネルギー 練習問題

練習問題

  1. 水素分子 (H₂) の結合エネルギーは436 kJ/molです。以下の反応のエネルギー変化を求めてください。

    H₂(g) → 2H(g)
  2. メタン (CH₄) の結合エネルギーは以下の通りです。
    • C-H 結合エネルギー: 414 kJ/mol
    メタン分子1個を構成する全てのC-H結合を切断するために必要なエネルギーは何kJ/molですか？
  3. 以下のデータを用いて、水の生成に関する反応の標準生成エンタルピー変化 (ΔH) を求めてください。
    • H-H 結合エネルギー: 436 kJ/mol
    • O=O 結合エネルギー: 498 kJ/mol
    • O-H 結合エネルギー: 463 kJ/mol

    H₂(g) + 1/2O₂(g) → H₂O(g)
  4. エタン (C₂H₆) におけるC-C結合の結合エネルギーを以下のデータを用いて求めてください。
    • C-H 結合エネルギー: 414 kJ/mol
    • O=O 結合エネルギー: 498 kJ/mol
    • C=O 結合エネルギー: 804 kJ/mol
    • O-H 結合エネルギー: 463 kJ/mol
    • 燃焼反応で得られるエンタルピー変化 (ΔH_c°): -1560 kJ/mol


    エタンの燃焼反応式:

    C₂H₆(g) + 7/2O₂(g) → 2CO₂(g) + 3H₂O(g)
  5. 以下のデータを使用して、窒素分子 (N₂) の結合エネルギーを求めてください。
    • N-H 結合エネルギー: 391 kJ/mol
    • H-H 結合エネルギー: 436 kJ/mol
    • アンモニア (NH₃) の生成エンタルピー変化 (ΔH_f°): -45.9 kJ/mol


    アンモニアの生成反応:

    1/2N₂(g) + 3/2H₂(g) → NH₃(g)
